## 💡 핵심 인사이트

### 1. 배송비 자체는 문제가 아닙니다
```
문제: 배송비 $15
실제 장벽: 

❌ 잘못된 접근:
"제품 $20 + 배송비 $15 = $35" 
→ 고객: "배송비가 제품값보다 비싸네?" → 이탈

✅ 올바른 접근:
"세트 $80 + 배송비 $15 = $95"
→ 고객: "배송비 비율 16%면 괜찮네!" → 구매
```

### 2. 객단가가 답입니다!
```
심리적 기준점:

배송비 비율 < 20% → 구매 확률 70%+
배송비 비율 < 15% → 구매 확률 85%+
배송비 비율 < 10% → 구매 확률 95%+

예시:
$20 제품 + $15 배송 = $35 (배송비 43%) → ❌ 이탈
$50 제품 + $15 배송 = $65 (배송비 23%) → ⚠️ 고민
$80 제품 + $15 배송 = $95 (배송비 16%) → ✅ 구매
$120 제품 + 무료 = $120 (배송비 0%) → 🎉 즉구매
```

## ⚠️ 피해야 할 실수 5가지

### ❌ 실수 1: 배송비가 제품보다 비쌈
```
나쁜 예:
- Sheet Mask $8 + 배송비 $15 = $23
- 고객: "말도 안 돼!" → 이탈

해결:
- 최소 주문액 설정 ($30+)
- 또는 세트만 판매
```

### ❌ 실수 2: 배송 기간 불명확
```
나쁜 예:
"배송: 7-21일" (너무 광범위)

좋은 예:
"K-Packet: 10-14일 (95% 확률)"
"Express: 5-7일 (100% 보장)"
```

### ❌ 실수 3: 숨겨진 비용
```
나쁜 예:
"무료 배송!" → 체크아웃에서 "관세 $20"
→ 신뢰도 폭락

좋은 예:
"배송비 $15 (관세 포함)"
또는 "DDP (세금 포함 배송)"
```

### ❌ 실수 4: 배송 추적 불가
```
나쁜 예:
"발송했습니다!" → 2주간 소식 없음
→ 고객 불안 → CS 폭주

좋은 예:
- 실시간 추적 번호
- 배송 단계별 이메일 알림
- 예상 도착일 업데이트
```

### ❌ 실수 5: 반품 정책 모호
```
나쁜 예:
"반품 가능" (구체적 설명 없음)
→ 반품비 $30 청구 → 분쟁

좋은 예:
"파손/불량만 교환 (사진 증거 필요)"
"개봉 후 반품 불가"
"왕복 배송비 고객 부담"
```
