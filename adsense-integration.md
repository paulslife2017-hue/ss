## 🔑 AdSense 광고 코드 받기

### 1단계: AdSense에서 광고 코드 복사
1. https://adsense.google.com 접속
2. 왼쪽 메뉴에서 **"광고"** 클릭
3. **"광고 단위 기준"** 선택
4. **"디스플레이 광고"** 생성 클릭
5. 광고 이름 입력: "Seoul Zen - Article Top"
6. **광고 코드 복사** (다음과 같은 형태):

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
     crossorigin="anonymous"></script>
<ins class="adsbygoogle"
     style="display:block"
     data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
     data-ad-slot="1234567890"
     data-ad-format="auto"
     data-full-width-responsive="true"></ins>
<script>
     (adsbygoogle = window.adsbygoogle || []).push({});
</script>
```

**중요**: `ca-pub-XXXXXXXXXXXXXXXX` 부분이 당신의 게시자 ID입니다!

## 🚀 빠른 시작: 자동 광고 (가장 쉬움)

### 자동 광고 코드만 추가하면 끝!

AdSense에서:
1. 왼쪽 메뉴 **"광고"** → **"사이트 기준"**
2. **seoulzen.com** 선택
3. 자동 광고 **"켜기"**
4. **자동 광고 코드** 복사 (다음 형태):

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
     crossorigin="anonymous"></script>
```

이 코드를 `<head>` 태그에 넣으면 Google이 자동으로 최적 위치에 광고 배치!
