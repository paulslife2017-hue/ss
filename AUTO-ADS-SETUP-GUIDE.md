## 🔴 중요: 지금 바로 해야 할 일 (5분)

코드만으로는 Auto Ads가 작동하지 않습니다!  
**반드시 Google AdSense 대시보드에서 수동으로 활성화**해야 합니다.

### 단계별 가이드:

#### 1️⃣ Google AdSense 로그인
```
URL: https://adsense.google.com
계정: ca-pub-6943282483618134 사용 중인 계정
```

#### 2️⃣ Auto Ads 설정으로 이동
1. 왼쪽 메뉴에서 **"Ads"** 클릭
2. **"By site"** 탭 선택
3. 사이트 목록에서 **"seoulzen.com"** 찾기
4. 오른쪽의 **연필 아이콘 (편집)** 클릭

#### 3️⃣ Auto Ads 켜기
1. **"Auto ads"** 토글을 **ON**으로 전환
2. 팝업이 나타나면 광고 형식 선택:

**권장 설정:**
```
✅ In-page ads (페이지 내 광고)
   → 콘텐츠 중간에 자연스럽게 삽입
   → 가장 수익성 높음
   → 강력 추천!

✅ Anchor ads (고정 광고 - 모바일)
   → 모바일 화면 하단에 고정
   → 스크롤해도 항상 표시
   → 모바일 수익 증가

⚠️  Vignette ads (전면 광고)
   → 페이지 전환 시 전체 화면 광고
   → 수익은 높지만 사용자 불편
   → 첫 2주는 끄는 것 추천
   → 나중에 테스트해볼 수 있음
```

#### 4️⃣ 광고 로드 설정
```
광고 로드 강도 선택:
  [ ] Low (적게)    - 사용자 경험 최우선
  [✓] Medium (중간) - 권장! 균형잡힌 설정
  [ ] High (많이)   - 최대 수익 (신중히)

첫 2주는 "Medium" 권장
→ 이탈률 확인 후 조정
```

#### 5️⃣ 적용하기
1. **"Apply to site"** 버튼 클릭
2. 확인 메시지: "Auto ads will be activated in 24-48 hours"
3. ✅ 완료!

## 📈 성과 모니터링

### AdSense 대시보드 확인

**일일 체크 (첫 2주):**
```
1. https://adsense.google.com → Reports
2. 확인할 지표:
   - Page RPM (페이지당 수익)
   - CTR (클릭률)
   - Impressions (노출 수)
   - Manual vs Auto Ads 비교
```

**주간 체크:**
```
Google Analytics (G-4DH40QL7TS):
- Bounce Rate (이탈률) - 증가하면 광고 줄이기
- Avg. Session Duration - 감소하면 조정
- Pages per Session - 유지 확인
```

### 최적화 기준:

#### ✅ 좋은 신호:
```
✅ Page RPM 증가 (+10-30%)
✅ 이탈률 유지 (<60%)
✅ 세션 시간 유지 (>2분)
✅ 사용자 불만 없음
```

#### ⚠️ 조정 필요:
```
⚠️  이탈률 급증 (+10% 이상)
⚠️  세션 시간 감소 (-20% 이상)
⚠️  Page RPM 감소
⚠️  사용자 불만 댓글

→ AdSense에서 광고 로드를 "Medium" → "Low"로 조정
```

## ❓ 문제 해결 (FAQ)

### Q1: Auto Ads가 보이지 않아요
```
A: 24-48시간 대기 필요
   - AdSense 대시보드에서 ON 되어있는지 확인
   - 브라우저 캐시 삭제 후 재확인
   - 시크릿 모드에서 테스트
```

### Q2: 광고가 너무 많이 나와요
```
A: AdSense → Auto ads → Ad load
   - "High" → "Medium" 또는 "Low"로 변경
   - Vignette ads 끄기
   - 24시간 후 재확인
```

### Q3: 수익이 오히려 줄었어요
```
A: 첫 2주는 학습 기간
   - Google AI가 최적 위치 찾는 중
   - 3-4주 후 다시 확인
   - 이탈률이 급증했다면 광고 줄이기
```

### Q4: 모바일에서 Anchor 광고가 거슬려요
```
A: AdSense → Auto ads → Settings
   - Anchor ads 끄기
   - 또는 위치 변경 (하단 → 상단)
   - In-page ads만 사용
```

### Q5: 수동 광고랑 겹쳐요
```
A: Google이 자동으로 조정
   - 수동 광고 위치 피해서 Auto Ads 배치
   - 겹치는 경우는 거의 없음
   - 겹친다면 수동 광고 일부 제거 고려
```
