```
1단계 (0-3개월): Amazon FBA (미국 창고) - 테스트 & 검증
   └─ 투자: $3,000-5,000 / 월수익 목표: $1,000-3,000
```

## 📊 3가지 판매 모델 완벽 비교

### Model 1: 🇺🇸 Amazon FBA (미국 창고 보관)

#### ✅ 장점
1. **빠른 배송** (Prime 2일 배송 → 전환율 2-3배 ↑)
2. **Amazon 신뢰도** 활용 (미국인들 Amazon 선호)
3. **FBA가 배송/CS 처리** (시간 절약)
4. **검색 노출 유리** (Amazon SEO 자동)
5. **반품 쉬움** → 구매 장벽 낮음

#### ❌ 단점
1. **초기 재고 투자 필요** ($3,000-10,000)
2. **FBA 수수료 높음** (판매가의 30-40%)
3. **재고 리스크** (안 팔리면 손실)
4. **경쟁 심함** (중국 셀러들 가격 경쟁)
5. **Amazon 정책 변동 리스크**

#### 💰 수익 계산 예시

**상품**: COSRX Advanced Snail 96 Mucin Power Essence (100ml)

| 항목 | 금액 |
|------|------|
| 한국 도매가 | ₩15,000 ($11) |
| 미국 운송비 (대량) | $2 |
| Amazon FBA 수수료 (30%) | $9 |
| **총 비용** | **$22** |
| **Amazon 판매가** | **$30** |
| **순이익** | **$8 (27%)** |

**월 판매 100개 시**:
- 매출: $3,000
- 순이익: $800
- 초기 재고 투자: $2,200 (100개)

### Model 2: 🇰🇷 한국 직배송 (Dropshipping/직접 배송)

#### ✅ 장점
1. **초기 투자 거의 0원** (재고 없음)
2. **다양한 상품 판매 가능** (재고 부담 없어서)
3. **마진율 높음** (중간 수수료 없음)
4. **독점 상품 가능** (미국에 없는 최신 제품)
5. **재고 리스크 0**

#### ❌ 단점
1. **배송 느림** (7-14일) → 전환율 낮음
2. **배송비 비쌈** ($15-25/개)
3. **관세 문제** (고객이 추가 비용 부담 가능)
4. **반품 복잡** (국제 배송)
5. **신뢰도 낮음** (미국 소비자들 꺼림)

#### 💰 수익 계산 예시

**상품**: 최신 K-Beauty Serum (한국 독점 제품)

| 국제 배송비 | $18 |

| **총 비용** | **$35** |

### Model 3: 🏪 자체 쇼핑몰 + 정기구독
