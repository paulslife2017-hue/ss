## 🚀 Google AdSense 신청 방법

### Step 1: AdSense 계정 생성
```
1. https://www.google.com/adsense 방문
2. Gmail 계정으로 로그인
3. "시작하기" 클릭
4. 웹사이트 URL 입력 (블로그 주소)
5. 이메일 수신 동의
```

### Step 2: 사이트 연결
```
1. AdSense에서 제공하는 코드 복사
2. 블로그 <head> 태그 사이에 삽입
3. "완료" 클릭하여 확인 요청
```

### Step 3: 승인 대기
```
- 일반적으로 1-2주 소요
- 승인 후 광고 게재 시작
- 초기 수익 확인까지 추가 1-2주
```

### Step 4: 광고 배치
```
승인 후 다음 위치에 광고 삽입:
1. 헤더 배너 (728x90)
2. 인피드 광고 (반응형)
3. 사이드바 광고 x2 (300x250)
4. 본문 광고 x3 (반응형)
5. 앵커 광고 (하단 고정)
```
