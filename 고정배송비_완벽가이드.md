```

- 미국 $0-49: $15

- 캐나다 $0-49: $18

- 캐나다 $100+: 무료
- 유럽 $0-49: $22
- ...
→ 머리 아파 죽음 🤯
```

### 2. 시스템 구축 5분

```
고정 $10:
if (checkout) {
  shippingFee = 10;
}

단계별:
if (country === 'US') {
  if (total < 50) shippingFee = 15;
  else if (total < 100) shippingFee = 8;
  else shippingFee = 0;
} else if (country === 'CA') {
  if (total < 50) shippingFee = 18;
  ...
} else if ...
→ 100줄 코드, 1주일 작업

```

→ 고객: "아 $10이구나!" ✅
